# Pocket 2048

Pocket 2048 is a premium-feeling mobile portrait take on the classic swipe-merge puzzle. The game keeps the original 4x4 number-merging loop, removes almost all chrome, and leans on soft glassy tiles, confident motion, and a calm native-app feel instead of loud arcade styling. The art direction is achievable without heavy gameplay assets because the runtime can be built from code-driven shapes, gradients, and typography, while the listing icon and hero can be purpose-made with light AI generation or bespoke compositing.

- Best platform: mobile portrait
- Additional supported platforms: desktop, because keyboard input maps cleanly and does not require layout compromises
- Gameplay family: UI / 2D board puzzle
- Game meta: Single Loop
- Input: Direct Touch on mobile portrait, Keyboard / Mouse on desktop
- Game category: Games / Puzzle / Number puzzle
- Reference games: 2048, Threes, and Stackdrop for Playdrop multi-surface packaging quality rather than mechanics
