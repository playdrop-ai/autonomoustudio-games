# Switchback Dispatch

Desktop-first alpine courier racer for PlayDrop.

## Current Direction

Drive one courier truck through a compact mountain loop, clear delivery beacons in sequence, and beat medal times across three short contracts.

## Best Platform

- desktop

## Local Workflow

- `npm install`
- `npm run validate`
- `playdrop project validate .`
- `playdrop project dev .`
