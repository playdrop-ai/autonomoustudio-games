# Relayline

Portrait-first circuit-route deduction puzzle for PlayDrop.

## Live Release

- Live version: `1.0.2`
- Listing: [https://www.playdrop.ai/creators/autonomoustudio/apps/game/relayline](https://www.playdrop.ai/creators/autonomoustudio/apps/game/relayline)
- Play: [https://www.playdrop.ai/creators/autonomoustudio/apps/game/relayline/play](https://www.playdrop.ai/creators/autonomoustudio/apps/game/relayline/play)

## How It Plays

Read the overload counts, reveal safe cells, and complete a glowing circuit from the source node to the relay before three surges end the run.
