# Glowknot - SIMPLIFY v1

## Core Promise In One Sentence

Fire glowing lantern shots into a hanging festival canopy, sever the right support clusters, and drop whole sections before the sinking line reaches the launcher.

## Differentiator That Survived Simplification

Visible anchor-cut canopy collapses remain the whole point. The game is still a bubble shooter, but the board is explicitly staged around a few obvious silk anchors so the most satisfying move is a dramatic detached drop, not generic color cleanup.

## Best Platform

`mobile portrait`

## Additional Supported Platforms That Still Survive

- `desktop`

## Final Gameplay Family

Single-screen hanging-cluster bubble shooter.

## Final Game Meta

Endless score chase.

## Final Input Model Per Supported Platform

- Mobile portrait: drag to aim, release to fire.
- Desktop: move to aim, click to fire.

## Final HUD And Key Screens

### Active Gameplay HUD

- score
- sinking-line danger indicator
- current shot
- reserve shot

### Key Screens

- start bottom sheet over the live board
- gameplay
- game-over bottom sheet over the frozen board

## Final Art And Asset Scope

- bespoke 2D lantern discs with soft glow and clean outlines
- three visible top anchors and dark silk ropes
- gradient night backdrop with simple silhouette depth
- particle bursts for pops and detached drops
- no character art, no 3D scene, no heavy texture pipeline

## Not In v1

- special lantern power pieces
- story framing or world map
- level progression or authored campaign stages
- achievements, leaderboards, or account-backed progression
- extra queue beyond current shot plus reserve shot
- daily challenge mode
- multiplayer
- settings screen beyond mute if it is trivially cheap
- elaborate painted backdrop if the simpler gradient-and-silhouette backdrop already looks shippable

## Why This Is Still The Smallest Strong v1

Cutting the visible anchors or reserve shot would collapse the concept into a generic bubble shooter. Keeping those two elements while cutting everything else leaves the smallest version that still feels like `Glowknot` instead of a thin clone.

## Why This Still Feels Like A Real Game Instead Of A Demo

The run is endless, the sinking line keeps pressure high, reserve-shot planning creates decision depth, and detached canopy drops create obvious high-score mastery moments. A competent player still has meaningful replay reasons after the first successful clear.
