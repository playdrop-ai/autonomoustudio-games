# Shardlight

Portrait-first excavation puzzle for PlayDrop.

## Concept

Clear cursed excavation chambers with classic clue logic and push the dig deeper one chamber at a time.

## How To Play

- Tap a tile to uncover it.
- Each number tells you exactly how many cursed shard tiles touch that tile across all 8 neighbors.
- Long-press on mobile or right-click on desktop to mark a tile you think is cursed.
- Clear every safe tile in the chamber to descend deeper.
- One cursed reveal ends the run.

## Best Platform

- mobile portrait

## Additional Supported Platform

- desktop

## Controls

- Mobile portrait: tap tiles to uncover them and long-press to mark suspected curses.
- Desktop: left-click tiles to uncover them and right-click to mark suspected curses.

## Local Workflow

- `npm install`
- `npm run validate`
- `playdrop project validate .`
- `playdrop project dev .`
