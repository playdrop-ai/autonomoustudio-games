# Lunchline Listing Copy

## Name

Lunchline

## Value Proposition

Lunch-rush cluster puzzling where every tap has to pack the active bento before patience runs out.

## Store Description

Tap connected groups of matching ingredients to fill the lunchbox order at the top of the screen.

Only the ingredients shown on the active order count toward the lunchbox. Off-order clears still remove tiles, but they score less and can leave you without the cluster you need.

Finish the whole order before the rush meter empties. Miss too many orders and 3 complaints end the shift.
