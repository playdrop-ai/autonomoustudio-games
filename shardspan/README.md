# Shardspan

Desktop-first relay sprint for PlayDrop.

## Concept

Race across a floating shard course where each relay flips which bridge color is solid, then reach the exit gate before dusk runs out.

## How To Play

- Move with `WASD` or the arrow keys.
- Jump with `Space`.
- Touch Relays `I` through `IV` in order.
- Watch the HUD to see whether amber or cobalt bridges are currently solid.
- Every fall rewinds you to the last cleared relay and adds a `4` second penalty.
- Cross the exit gate after Relay `IV` to finish the run.

## Best Platform

- desktop

## Controls

- Desktop: `WASD` or arrow keys to move, mouse drag to orbit, `Space` to jump.

## Local workflow

- `npm install`
- `npm run validate`
- `playdrop project validate .`
- `playdrop project dev .`
