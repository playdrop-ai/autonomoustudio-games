# Shardspan - SIMPLIFY v1

## Core Promise In One Sentence

Sprint a compact floating relay course where each beacon flip makes the next bridge set real.

## Differentiator That Survived Simplification

Ordered beacon touches flip the world's solid path state, so route readability and jump execution stay fused into one mechanic.

## Best Platform

`desktop`

## Additional Supported Platforms That Still Survive

- None in v1.

## Final Gameplay Family

Single-course 3D phase-switch platformer.

## Final Game Meta

Short time-attack run with best-time replay value.

## Final Input Model Per Supported Platform

- Desktop: `WASD` or arrows to move, mouse drag to orbit camera, `space` to jump.

## Final HUD And Key Screens

### HUD

- Timer bar with numeric seconds remaining
- Relay progress `0/4` through `4/4`
- Next-beacon direction cue

### Key Screens

- Start overlay
- Active gameplay
- End overlay with retry

## Final Art And Asset Scope

- Reuse the starter kit character and main platform pieces.
- Recolor bridge families into amber and cobalt.
- Add only bespoke beacon markers, finish gate treatment, and lightweight pulse/shimmer VFX.
- Keep one dusk sky lighting setup for the whole run.

## Final Course Scope

- One compact desktop course
- Four relay beacons in fixed order
- One exit gate after the fourth beacon
- Respawn at the last cleared beacon with a time penalty on falls

## Not In v1

- Mobile support
- Multiple levels
- Alternate routes that branch the player away from the highlighted beacon
- Moving hazards
- Enemies or combat
- Side collectibles
- Narrative framing
- Character reskin or new avatar rig
- Online leaderboards
- Photo mode, accessibility submenu, or settings screen beyond what the starter already needs

## What Got Cut From The Spec

- Any support surface beyond desktop
- Any extra traversal system beyond move, jump, camera orbit, and beacon flipping
- Decorative side islands that do not affect the main route
- Optional challenge pickups or combo scoring
