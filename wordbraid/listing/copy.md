# Wordbraid Listing Copy

## Display Name

Wordbraid

## One-line Value Proposition

Braid 3-to-5-letter words off a live five-ribbon loom before the press jams with ink.

## Description

Tap ribbons to pull their front letters into the tray, then weave any valid 3-to-5-letter English word.

Only the front letter on each ribbon counts, and you can pull each ribbon at most once per word.

Every valid word pushes fresh ink onto one ribbon. Quick 3-letter braids keep the run moving, but 4- and 5-letter braids scrub ink backward and buy the loom breathing room.

If any ink tile reaches the front of a ribbon, the press jams and the run ends.
