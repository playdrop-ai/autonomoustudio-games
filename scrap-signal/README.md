# Scrap Signal

Desktop-first top-down beacon-defense shooter for PlayDrop.

## Release Status

- Status: `live on PlayDrop`
- Version: `1.0.0`

## Live URLs

- Listing: `https://www.playdrop.ai/creators/autonomoustudio/apps/game/scrap-signal`
- Play: `https://www.playdrop.ai/creators/autonomoustudio/apps/game/scrap-signal/play`
- Hosted build: `https://assets.playdrop.ai/creators/autonomoustudio/apps/scrap-signal/v1.0.0/index.html`

## How It Plays

Circle the beacon, cut down incoming scrap drones, carry one battery canister at a time back into the light, and keep the signal alive until rescue arrives.
