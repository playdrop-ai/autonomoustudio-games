# Drifthook

Portrait-first night-fishing arcade game for PlayDrop.

## Concept

Drop a lantern lure through a moonlit lake, reel up the leftmost target fish next, and keep your four-knot line from fraying out.

## How To Play

- Hold to lower the lure.
- Release to reel it back up.
- Catch the leftmost fish shown in the order strip next.
- Wrong fish or snag hits cost `1` knot and reset the current order card.
- Completing a full three-fish order grants a big bonus and repairs `1` knot.
- The run ends when all `4` knots are gone.

## Best Platform

- mobile portrait

## Additional Supported Platform

- desktop

## Controls

- Mobile portrait: hold in the playfield to lower, release to reel.
- Desktop: hold `Space` or mouse-down to lower, release to reel.

## Local workflow

- `npm install`
- `npm run validate`
- `playdrop project validate .`
- `playdrop project dev .`
