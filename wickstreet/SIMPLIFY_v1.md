# Wickstreet - SIMPLIFY v1

## Core Promise In One Sentence

Carry the right glow core from the depot to the matching dark home before the timer burns out and the storm closes more streets.

## Differentiator That Survived Simplification

The game still lives on active-route plus next-route planning inside a mutable street grid. That planning loop is the reason it exists, so it stays.

## Best Platform

`mobile landscape`

## Additional Supported Platforms That Still Survive

- `desktop`

## Final Gameplay Family

Top-down courier route arcade.

## Final Game Meta

One endless score run with `3` blackout strikes, best-score persistence, and longest-streak persistence.

## Final Input Model Per Supported Platform

- Mobile landscape: drag anywhere on the playfield to steer the courier. Release to stop.
- Desktop: `WASD` or arrow keys to move. Mouse-hold steering stays as an alternate input only if it does not weaken keyboard feel.

## Final HUD And Key Screens

- Gameplay HUD:
  - score
  - `3` blackout strike pips
  - active request card with timer ring
  - next-request preview card
- Start overlay:
  - title
  - one-line rule
  - `Play` CTA
- Game-over overlay:
  - score
  - best
  - longest streak
  - `Retry` CTA

## Final Art And Asset Scope

- one compact neighborhood map
- one courier character
- `4` house silhouettes with distinct sigils
- one depot kiosk
- one flood-barrier family
- rain streaks, puddle reflections, and brief relight flashes
- no bespoke character animation beyond bob, blink, and delivery pulse

## Why The Simplified Version Is Still A Real Game

Even after the cuts, the player is still reading routes, anticipating the next request, and adapting to new barriers under a tightening timer. That loop has enough mastery and score-chase potential to support multi-minute runs instead of a one-clear demo.

## Not In v1

- multiple districts or alternate maps
- pedestrians, traffic, or moving enemies
- upgrades, currencies, unlock trees, or meta progression
- multiplayer, leaderboards, or live service features
- weather variants beyond the single storm ramp
- portrait support
- tutorial popups beyond one concise rule line on the start overlay
- delivery pickups outside the single central depot
- more than one active request at once
