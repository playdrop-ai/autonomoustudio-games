# Wickstreet

Landscape-first blackout courier arcade game for PlayDrop.

## Concept

Race a rain-slick courier across a dark block grid, grab the live glow core from the depot, and relight the exact matching house before the blackout timer burns out.

## How To Play

- Drag anywhere on the playfield to steer the courier.
- Step through the depot to collect the active glow core.
- Deliver that glow core to the house with the exact same sigil.
- Read the next-request card early so you can finish the current route while planning the return line.
- Every wrong delivery or timer miss adds `1` blackout strike.
- The run ends at `3` blackout strikes.

## Best Platform

- mobile landscape

## Additional Supported Platform

- desktop

## Controls

- Mobile landscape: drag anywhere on the playfield to steer toward your finger, then release to stop.
- Desktop: move with `WASD` or arrow keys, or hold the mouse on the playfield to steer toward the pointer. Press `Space` to start or retry.

## Local workflow

- `npm install`
- `npm run validate`
- `npm test`
- `playdrop project validate .`
- `playdrop project dev .`
