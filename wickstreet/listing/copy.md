# Wickstreet Store Copy

## Value Proposition

Race a tiny lamplighter through a rain-soaked block grid, cut the right route, and relight the whole neighborhood one exact delivery at a time.

## Listing Description

Drag anywhere on the playfield to steer the courier through the storm. Step through the depot to grab the active glow core, then deliver it to the house with the exact same sigil before the blackout timer burns out.

The next-request card always shows which home is queued next, so the best runs come from finishing the current route while already planning the return line. Every missed timer adds `1` blackout strike, and `3` strikes end the run.

Desktop support keeps the same loop with `WASD` or arrow keys.
