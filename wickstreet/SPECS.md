# Wickstreet - High-level Requirements

- Ship a landscape-first PlayDrop game that feels materially different from the studio's portrait-heavy catalog.
- Keep the core loop focused on avatar movement, route reading, and exact matching of glow cores to homes.
- Preserve a small, strong v1: one map, one courier, one storm system, one endless score chase.
- Support `mobile landscape` first and `desktop` second without compromising the primary surface.
- Keep gameplay edge to edge with minimal HUD and clear start/game-over flows.
- Require real mockups, real browser validation, real gameplay capture, PlayDrop-first listing media, live verification, README release info, and the X thread before calling the release done.
