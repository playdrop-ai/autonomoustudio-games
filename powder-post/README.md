# Powder Post

Landscape-first downhill delivery slalom for PlayDrop.

## How To Play

- Drag across the slope on mobile landscape.
- On desktop, use `A`, `D`, or the arrow keys.
- Match the active parcel crest in the HUD to the gate with the same shape and color.
- Correct gates build score and push the storm back uphill. Wrong gates, obstacle hits, and missed deliveries let the avalanche close in.
