# Powder Post Listing Copy

## Display Name

Powder Post

## One-line Value Proposition

Carve downhill, thread the right village gate, and keep the avalanche off your back.

## Description

Drag to carve across the slope. On desktop, `A`, `D`, and the arrow keys also steer.

Match the active parcel crest in the HUD to the gate with the same crest shape and color.

A correct gate extends your streak, scores bigger, and pushes the storm back uphill. A wrong gate, a bad obstacle hit, or too many missed deliveries lets the avalanche close the slope and ends the run.

Read the next parcel preview early so you can line up the next carve before the gate pair reaches you.
