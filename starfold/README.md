# Starfold

Portrait-first chain-reaction puzzle for PlayDrop.

## Concept

Swipe whole rows and columns of glowing sigils to weave cascading constellations before ash consumes the shrine.

## Best Platform

- mobile portrait

## Additional Supported Platform

- desktop

## Controls

- Mobile portrait: swipe directly across a row or column to shift that line by one cell.
- Desktop: click-drag across a row or column to shift that line by one cell.

## Local workflow

- `npm install`
- `npm run validate`
- `playdrop project validate .`
- `playdrop project dev .`
